# Compos CLI

[![English](https://img.shields.io/badge/lang-English-0A66C2)](README.md)
[![한국어](https://img.shields.io/badge/lang-%ED%95%9C%EA%B5%AD%EC%96%B4-BDBDBD)](README.ko.md)

Command-line client for Compos workspaces.

Use this distribution entry to download released `compos` CLI binaries built from the upstream source repository.

## Links

- Source repository: https://github.com/Conalog/compos
- Releases page: https://github.com/Conalog/publish/releases

## Install

Download the latest `compos/vX.Y.Z` release asset for your platform:

- `compos_<version>_linux_amd64.tar.gz`
- `compos_<version>_linux_arm64.tar.gz`
- `compos_<version>_macos_amd64.tar.gz`
- `compos_<version>_macos_arm64.tar.gz`
- `compos_<version>_windows_amd64.tar.gz`

Extract the archive and run:

```sh
bin/compos version
```

## Uninstall

Remove the extracted archive directory or delete the installed `compos` binary from your PATH.

## Notes

Release assets are the source of truth for published versions. This product entry intentionally does not duplicate a latest version number.
